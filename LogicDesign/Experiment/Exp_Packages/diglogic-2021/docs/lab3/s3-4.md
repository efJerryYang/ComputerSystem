可以将整个系统分为3个模块：

clk_div：用于时钟分频；

led_mem：用于存储LED灯依次显示的序列；

memory_w_r：用于控制存储器的读写，以及将读取回来的序列，显示到LED上；

   

#### 工作时序

 <center><img src="../s3-21.png" width = 700></center>

Reset：模块处于复位状态；

Not work：模块处于等待启动信号状态；

Write RAM：将要显示的LED灯序列写入RAM；

Read RAM：将RAM中的LED灯序列读取出来，显示到LED上；
